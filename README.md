# 🚀 Fleet Performance & Reliability Command Center (Power BI)

![Dashboard Preview] (Dashboard.png)

## 📌 Project Overview
This project is a custom-engineered, dark-mode Power BI dashboard designed to act as an operational Command Center for fleet managers. Moving away from standard native visuals, this build focuses on a premium SaaS-style UI/UX, prioritizing visual hierarchy and immediate risk identification.

## 🎯 The Business Problem
Fleet managers often struggle with data overload, making it difficult to identify which machines are draining maintenance budgets or causing operational bottlenecks. The goal of this dashboard was to segment the fleet and isolate the highest-risk assets instantly.

## 🛠️ Technical Highlights & Engineering
* **UI/UX Wireframing:** Built a custom background canvas using strict wireframing principles to lock visuals in place, prevent rendering bugs, and reduce eye strain (Dark Mode).
* **Tiered Asset Segmentation:** Grouped the fleet into Critical (Heavy Cranes/Telehandlers), Standard (Forklifts/Reach Trucks), and Support (Pallet Jacks/Utility Carts) tiers.
* **Dynamic KPIs & DAX:** Engineered DAX measures for calculating Month-over-Month percentage changes with dynamic up/down trend indicators.
* **Granular Matrix Design:** Customized the native matrix visual with inline data bars and stacked text for a clean, space-saving "ledger" view.
